# PaniDobro Brand Assets

- **Wordmark** (`app/static/svg/logo_wordmark_light.svg`) is reserved for light backgrounds only.
- **Emblem** (`app/static/svg/logo_mark.svg`) can be used in monochrome when needed.
- Avoid gradients and do not apply accent teal to the logo forms.
- Maintain clear space equivalent to the height of the emblem around all logo placements.
